# slush-xdc-pages-vue

> A multiple pages project for vue.

## Dependency
- [xdc-vue](https://github.com/x-xdc/xdc-vue/)
- [xdc-lint](https://github.com/x-xdc/xdc-lint/)
- [xdc-buble](https://github.com/x-xdc/xdc-buble/)

## Quick Start
```shell
npm i xdc -g

mkdir test-vue && cd test-vue
xdc init pages-vue
```
